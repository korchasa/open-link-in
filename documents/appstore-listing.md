## English (en-US)

- **Name (≤30):** Smart Links Opener
- **Subtitle (≤30):** Open links per-domain
- **Keywords (≤100):** browser,default browser,link,router,picker,domain,open in,chrome,safari,firefox,switch
- **Promotional text (≤170):** Send every link to the right browser automatically. Set a rule per domain — work links to one browser, personal to another — or pick on the fly.
- **Description:**
```
Smart Links Opener makes your Mac open each link in the browser you actually want.

Set it as your default browser, and instead of forcing every link into one app, it
routes links by domain: work sites to one browser, personal to another, a client's
app to a third. When a link has no rule yet, a clean picker lets you choose a browser
— and optionally remember the choice for that whole domain.

FEATURES
• Per-domain routing — pick which browser opens which site.
• Subdomain support — one rule for a domain covers all its subdomains.
• Quick picker for new links, with one-click "remember".
• Lives in the menu bar, out of your way — no Dock clutter.
• Matched links open instantly, without stealing focus.
• Native, lightweight, no tracking, no accounts, works offline.

Open source: the code is available under GPLv3 at github.com/korchasa/open-link-in.
This App Store build is the convenient, signed, auto-updating version.
```
